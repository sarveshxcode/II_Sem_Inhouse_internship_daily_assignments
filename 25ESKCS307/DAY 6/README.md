# Product Explorer

Responsive Bootstrap product directory using the DummyJSON Products API.

## API
https://dummyjson.com/products?limit=100

## Included Features
- Working fetch() call
- Loading state
- Error handling with .catch()
- Retry button
- Bootstrap responsive card grid
- Live product search
- Category filter
- Sort by name, price, or rating
- Product count badge
- Refresh button
- Product details modal
- Product images
- Discount badges
- Ratings
- Prices and stock display
- Responsive mobile design
- Custom CSS styling

## Run
Open `index.html` in your browser, or use VS Code Live Server.

Internet access is required for the API, Bootstrap, and Bootstrap Icons.


## Fix in this version
- Replaced native category and sort `<select>` controls with custom Bootstrap dropdowns.
- Dropdown text and backgrounds remain visible on Chrome, Windows, Android, and mobile screens.
